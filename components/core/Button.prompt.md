**Button** — the soft, hairline-bordered button used across the app; use it for actions and the square calendar month-nav control.

```jsx
<Button variant="primary">저장</Button>
<Button variant="default" size="icon">‹</Button>
<Button variant="ghost" size="sm">취소</Button>
```

Variants: `default` (white, hairline, tinted hover), `primary` (purple solid), `pink` (해인 solid), `ghost`. Use `size="icon"` for the 30×30 nav button, and `person="junghwa" | "haein"` to bind the accent hue.
