**ProgressBar** — slim recessed track with a hue-gradient fill, used for each person's monthly dosage rate.

```jsx
<ProgressBar person="junghwa" label="정화 복용률" value={82} />
<ProgressBar person="haein" label="해인 복용률" value={64} />
```

`value` 0–100; `person` sets the gradient hue; `label` / `showPct` control the header row.
